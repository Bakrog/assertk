fun <T> Assert<Array<T>>.each(f: (Assert<T>) -> Unit): Unit

Platform and version requirements: Common

Asserts on each item in the array. The given lambda will be run for each item.

assertThat(arrayOf("one", "two")).each {
  it.hasLength(3)
}
